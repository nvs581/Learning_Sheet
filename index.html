HTML5 Cheat Sheet

HTML Element Use / Meaning Example Usage
<!DOCTYPE> Defines the document type and HTML version (HTML5). <!DOCTYPE html>
<html> Root element of an HTML document. <html></html>
<head> Contains meta-information about the HTML document. <head></head>
<title> Defines the title of the document (appears on the browser tab). <title>Page Title</title>
<meta> Provides metadata about the document (e.g., charset, viewport). <meta charset="UTF-8">
<body> Defines the document's body, where visible content is placed. <body></body>
<h1> to <h6> Defines HTML headings (h1 is the largest, h6 is the smallest). <h1>Heading 1</h1>
<p> Defines a paragraph. <p>This is a paragraph.</p>
<a> Defines a hyperlink. <a href="https://example.com">Link</a>
<img> Embeds an image into the document. <img src="image.jpg" alt="Description">
<ul> / <ol> Defines unordered (ul) or ordered (ol) lists. <ul><li>Item</li></ul>
<div> Defines a division or section in an HTML document. <div>Content</div>
<span> Defines a section in a document (inline-level). <span>Inline content</span>
<footer> Defines a footer for a document or section. <footer>Footer content</footer>
Semantic Elements
<header> Defines a header for a document or section. <header><h1>My Website</h1></header>
<nav> Defines a set of navigation links. <nav><ul><li><a href="#">Home</a></li></ul></nav>
<section> Defines a section in a document. <section><h2>About Us</h2><p>Content here.</p></section>
<article> Defines independent, self-contained content. <article><h2>Blog Post</h2><p>Post content.</p></article>
<aside> Defines content aside from the content it is placed in. <aside>Sidebar content</aside>
<main> Specifies the main content of a document. <main>Main content here.</main>
<figure> & <figcaption> Groups media content with a caption. <figure><img src="image.jpg" alt="Description"><figcaption>Caption text</figcaption></figure>
Media Elements
<audio> Embeds sound content. <audio controls><source src="audio.mp3" type="audio/mpeg"></audio>
<video> Embeds video content. <video controls><source src="video.mp4" type="video/mp4"></video>
<canvas> Used to draw graphics via scripting (usually JavaScript). <canvas id="myCanvas" width="200" height="100"></canvas>
Form Elements
<form> Defines an HTML form for user input. <form action="/submit" method="post"><input type="text" name="name"></form>
<input> Defines an input control. <input type="email" name="email">
<textarea> Defines a multiline text input control. <textarea name="message"></textarea>
<select> Defines a drop-down list. <select name="options"><option value="1">Option 1</option></select>
<button> Defines a clickable button. <button type="submit">Submit</button>
Graphics Elements
<svg> Defines Scalable Vector Graphics. <svg width="100" height="100"><circle cx="50" cy="50" r="40" fill="red"></circle></svg>
Interactive Elements
<details> & <summary> Creates an interactive widget that the user can open and close. <details><summary>More Info</summary><p>Additional details here.</p></details>
<progress> Represents the progress of a task. <progress value="70" max="100">70%</progress>
<meter> Defines a scalar measurement within a known range. <meter value="2" min="0" max="10">2 out of 10</meter>
Text Content Elements
<mark> Defines marked or highlighted text. <p>This is <mark>important</mark> text.</p>
<time> Defines a date/time. <time datetime="2023-10-05">October 5, 2023</time>
<code> Defines a piece of computer code. <p>Use the <code>printf()</code> function.</p>
<pre> Defines preformatted text. <pre>Line 1 Line 2</pre>
